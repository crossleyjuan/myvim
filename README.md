Install
=======


git clonse git@github.com:crossleyjuan/myvim
ln -s .vimrc ~/.vimrc

git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim

then open vim and execute this:

:BundleInstall

restart and enjoy
